tarea_poo_visitas
Sistema de escritorio para registrar visitantes de una oficina, desarrollado en Python con Tkinter.
Cómo ejecutar

Clona el repositorio
Entra a la carpeta visitas_app
Ejecuta el programa con python main.py

Funcionalidades

Registrar visitantes
Ver lista de visitantes
Eliminar visitantes
Limpiar formulario

Requisitos

Python 3
No requiere librerías externas
